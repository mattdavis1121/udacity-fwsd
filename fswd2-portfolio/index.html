Reddit Blackjack Bot

The Reddit Blackjack Bot will deal a hand of blackjack to any Reddit user in any part of the site. Users summon the bot by mentioning it by name and issuing any of the allowed commands. Currently, the bot can only respond to deal, hit, and stay commands. Planned upgrades include split, double down, variable betting, high score tracking, and user achievements.
